# ИИ для учёбы

Цель: **понять**, а не “получить ответ любой ценой”.

## Безопасный учебный промпт
```text
Будь преподавателем.
Объясни <тему> простыми словами.
Потом задай 5 вопросов, чтобы проверить понимание.
Если не уверен в факте — пиши UNKNOWN.
```

## Лучшие результаты
- Конспект + ключевые термины
- Тренировочные вопросы с ответами
- План подготовки на 7–14 дней
