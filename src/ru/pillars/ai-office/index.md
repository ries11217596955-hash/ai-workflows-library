# ИИ для офиса

Базовое правило: **ИИ пишет черновик, ты проверяешь**.

## Универсальный офис‑промпт
```text
Роль: операционный ассистент.
Цель: сделать <результат>.
Факты (вход): <список пунктов>.
Ограничения: без выдуманных данных, кратко, по делу.
Формат: структурированный markdown с заголовками.
В конце добавь раздел “Проверки”: что мне нужно перепроверить?
```

## Лучше всего работает
- Превращать хаотичные заметки в **план** (шаги, ответственные, сроки)
- Делать **таблицы** (сравнение, график, чек‑лист)
- Писать **письма** с темой + чётким действием (CTA)
