# ИИ для письма

Это хаб: практичные паттерны + промпты “копипаст”.

## Что ты получишь
- Простой процесс написания и редактуры.
- 7 переиспользуемых паттернов для писем, документов и постов.
- Промпты (минимальный + расширенный).

## Процесс из 5 шагов (стабильный)
1) Определи **цель** и **аудиторию**.
2) Дай **входные данные** (факты, ссылки, ограничения).
3) Попроси сначала **структуру** (план).
4) Сгенерируй черновик.
5) Прогони **проверку качества** (см.: [Чек-лист качества](/ru/tools/quality-checklist/)).

## Основные паттерны

### 1) Сначала план (потом текст)
**Промпт**
```text
Составь план для: <тема>.
Аудитория: <кто>.
Цель: <какой результат>.
Ограничения: <объём, тон, что обязательно включить>.
Выход: нумерованный план (H2/H3).
```

### 2) Переписать с ограничениями
**Промпт**
```text
Перепиши текст ниже.
Смысл сохранить, ясность улучшить.
Ограничения: <тон, объём, уровень>.
Термины не менять: <список терминов>.
Текст:
<ВСТАВЬ>
```

### 3) Проверка корректности (факт-гард)
**Промпт**
```text
Проверь текст на возможные фактические ошибки и неоднозначные утверждения.
Пометь:
- [CHECK] требует проверки
- [OK] вероятно корректно
Затем предложи более безопасные формулировки без непроверенных фактов.
Текст:
<ВСТАВЬ>
```

### 4) Переписать в стиль (без добавления фактов)
**Промпт**
```text
Перепиши в стиле: <официально / дружелюбно / строго / технично>.
Смысл не менять, новые факты не добавлять.
Текст:
<ВСТАВЬ>
```

### 5) Саммари для решения
**Промпт**
```text
Сделай саммари для руководителя.
Выход:
1) 5 буллетов
2) Риски / неизвестности
3) Рекомендованный следующий шаг (1)
Текст:
<ВСТАВЬ>
```

### 6) Превратить заметки в документ
**Промпт**
```text
Преврати заметки в структурированный документ.
Формат:
- Заголовок
- Контекст
- Ключевые пункты
- Открытые вопросы
- Список действий (Ответственный / Срок / Статус)
Заметки:
<ВСТАВЬ>
```

### 7) Варианты A/B/C
**Промпт**
```text
Сделай 3 варианта одного сообщения:
A) коротко и прямо
B) нейтрально и официально
C) дружелюбно и убедительно
Ограничения: <объём, что обязательно включить>.
Черновик идеи:
<ВСТАВЬ>
```

## Стартовые пакеты (копипаст)

### Минимальный запрос
```text
Задача: <одно предложение>.
Аудитория: <кто>.
Ограничения: <тон/объём/что обязательно>.
Формат ответа: <буллеты / таблица / разделы>.
```

### Расширенный запрос (лучше контроль)
```text
Задача: <задача>.
Контекст: <фон>.
Входные данные: <факты/цифры, которые ты даёшь>.
Ограничения: <приватность, тон, объём>.
Выход:
1) Черновик
2) Допущения (если есть)
3) Какие проверки качества сделаны
```
