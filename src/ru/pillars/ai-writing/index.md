# ИИ для текста (надёжный результат)

Используй, когда нужен **чистый текст** (письма, документы, лендинги) без выдумок и воды.

## Метод 4 шагов
1) **Определи результат** (формат, объём, аудитория, тон).  
2) **Дай входные данные** (факты, ограничения, примеры).  
3) **Заставь структуру** (заголовки, списки, таблицы).  
4) **Сделай проверку качества** (ниже).

## “Контракт писателя” (копипаст)
```text
Ты — мой помощник по тексту.
Задача: <что сделать>.
Аудитория: <кто читает>.
Тон: <нейтральный / дружелюбный / строгий>.
Ограничения: <что обязательно / что запрещено>.
Источники: если не уверен — пиши "UNKNOWN", не выдумывай.
Формат ответа: <markdown / письмо / чек‑лист / таблица>.
```

## Проверка качества
- Есть ли выдуманные факты/цифры? Если да → убрать или пометить UNKNOWN.
- Читается ли структура (заголовки, пункты)?
- Совпадает ли тон и объём с запросом?
