Опора для создания ясных и согласованных текстов с AI — от брифа до готового результата.

## Область
Входит: брифы, планы, черновики, переписывание, тональность, структурированные документы.
Не входит: глубокое доменное исследование без источников, а также юридические/медицинские рекомендации.

## Принципы
- **Сначала контракт**: формат, аудитория, ограничения, критерии “готово”.
- **Структура важнее стиля**: план до черновика.
- **Допущения помечаются**: не прятать нехватку данных.
- **Одна секция = одна цель**: не смешивать намерения.
- **Согласованность важна**: термины, время, голос, форматирование.
- **Редактура проходами**: смысл → структура → язык → полировка.

## Стандартные потоки
### 1) Бриф → План → Черновик
1. Бриф: цель, аудитория, ограничения, ключевые пункты, примеры.
2. План: заголовки + краткий смысл каждого блока.
3. Черновик: заполнить секции, не выходить за scope.
4. QA: прогнать чек-лист, закрыть пробелы.

### 2) Переписать / улучшить ясность
1. Сохранить смысл (без смыслового дрейфа).
2. Упростить структуру (короче предложения, чище абзацы).
3. Нормализовать термины и тон.
4. Финальный QA.
