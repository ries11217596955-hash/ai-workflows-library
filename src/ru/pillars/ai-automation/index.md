# ИИ для автоматизации (база)

Думай **конвейером**, а не “одним промптом”.

## Шаблон конвейера
1) Вход (что есть)  
2) Трансформация (что делает ИИ)  
3) Проверка (контроль качества)  
4) Выход (куда складываем)  
5) Лог (что фиксируем)

## Промпт (копипаст)
```text
Ты — архитектор процессов.
Есть повторяющаяся задача: <задача>.
Спроектируй конвейер: Вход → Трансформация → Проверка → Выход → Лог.
Для каждого шага укажи: нужные инструменты, типовые сбои и быстрые фиксы.
Сделай максимально просто и автоматизируемо.
```
