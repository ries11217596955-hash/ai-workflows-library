# Таблица сравнения

**Цель:** Сделать таблицу, которая помогает выбрать (инструменты, сервисы, тарифы).

## Что нужно на вход (inputs)
- Список вариантов (3–10)
- Критерии (5–12) или попросить ИИ предложить
- Ограничения (бюджет, must‑have)

## Готовые промпты (копипаст)
**Промпт 1 — Таблица + оценка**

```text
Сделай таблицу сравнения.
Варианты:
<<<OPTIONS>>>
Критерии:
<<<CRITERIA>>>
Выход:
1) Таблица: критерии × варианты с короткими заметками
2) Оценка (0–5) по каждому критерию
3) Рекомендация с объяснением
Ограничения:
<<<CONSTRAINTS>>>
```

**Промпт 2 — Предложить критерии**

```text
Предложи 10 критериев выбора между этими вариантами. Критерии сделай измеримыми.
Варианты:
<<<OPTIONS>>>
```

**Промпт 3 — Decision memo**

```text
Напиши decision memo на 1 страницу: контекст, варианты, оценка, решение, риски.
Используй таблицу:
<<<TABLE>>>
```


## Проверка качества (30 секунд)
- Критерии отражают реальные ограничения
- Заметки основаны на фактах
- Внятные компромиссы

## Типовые ошибки
- Слишком расплывчатые критерии
- Нет объяснения tradeoff
- Оценки без причин
