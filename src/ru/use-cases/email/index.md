# Деловое письмо

**Цель:** Написать письмо, которое понятно и на которое отвечают.

## Что нужно на вход (inputs)
- Кто ты
- Кто получатель (роль)
- Контекст (1–3 пункта)
- Что хочешь получить (1 фраза)

## Готовые промпты (копипаст)
**Промпт 1 — Черновик письма**

```text
Напиши деловое письмо.
Тон: нейтральный, уважительный.
Цель: <цель>.
Ограничения: коротко, без воды, одно чёткое действие (CTA).
Выход: Тема + Текст письма.
Контекст:
<<<CONTEXT>>>
```

**Промпт 2 — Фоллоу‑ап**

```text
Напиши вежливый фоллоу‑ап к предыдущему письму. До 80 слов. Добавь один вопрос.
Предыдущее письмо:
<<<EMAIL>>>
```

**Промпт 3 — Письмо про границы работ**

```text
Перепиши письмо так, чтобы чётко определить объём работ и предотвратить бесконечные правки.
Добавь: что входит/не входит и следующий шаг.
Письмо:
<<<EMAIL>>>
```


## Проверка качества (30 секунд)
- Тема соответствует просьбе
- Одно действие (CTA)
- Нет расплывчатости
- Факты верны

## Типовые ошибки
- Несколько просьб в одном письме
- Нет CTA
- Слишком длинно
