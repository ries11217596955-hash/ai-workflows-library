# Структура презентации

**Цель:** Сделать понятную структуру презентации (10–12 слайдов).

## Что нужно на вход (inputs)
- Аудитория
- Цель
- Ключевой месседж (1 фраза)
- 3–5 опорных тезисов
- Данные, которые обязательно включить

## Готовые промпты (копипаст)
**Промпт 1 — Структура презентации**

```text
Сделай структуру из 10–12 слайдов.
Аудитория: <аудитория>
Цель: <цель>
Ключевой месседж: <1 фраза>
Для каждого слайда: заголовок + 3 пункта + заметка спикера.
Ограничения: кратко, без выдуманных данных.
Вход:
<<<INPUTS>>>
```

**Промпт 2 — Один слайд — резюме**

```text
Сделай один слайд executive summary: проблема, решение, эффект, следующий шаг.
Контекст:
<<<CONTEXT>>>
```

**Промпт 3 — Сториборд**

```text
Преврати структуру в сториборд: связный рассказ по слайдам и переходы.
Структура:
<<<OUTLINE>>>
```


## Проверка качества (30 секунд)
- У каждого слайда одна цель
- Данные не выдуманы
- Поток логичный
- В конце — следующий шаг

## Типовые ошибки
- Слишком много пунктов
- Нет истории
- Слайды в случайном порядке
