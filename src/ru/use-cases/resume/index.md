# Резюме под вакансию

**Цель:** Сделать резюме под конкретную вакансию + пункты достижений под требования.

## Что нужно на вход (inputs)
- Твоё текущее резюме (текстом)
- Описание вакансии (текстом)
- 2–3 измеримых достижения (если есть)

## Готовые промпты (копипаст)
**Промпт 1 — Переписать резюме**

```text
Перепиши моё резюме под эту вакансию. Факты сохраняй. Ничего не выдумывай (компании/даты/сертификаты).
Формат:
1) Summary (3 строки)
2) Skills (пункты)
3) Опыт (пункты в стиле STAR)
4) Список ключевых слов
Резюме:
<<<RESUME>>>
Вакансия:
<<<JD>>>
```

**Промпт 2 — Достижения**

```text
Из раздела опыта перепиши пункты так, чтобы были измеримые результаты. Факты не добавляй.
Выход: 8–12 пунктов.
Текст:
<<<TEXT>>>
```

**Промпт 3 — Проверка на соответствие**

```text
Сравни резюме с вакансией.
Выход:
- Чего не хватает по ключевым словам
- Слабые места
- 5 конкретных улучшений (без воды)
Резюме:
<<<RESUME>>>
Вакансия:
<<<JD>>>
```


## Проверка качества (30 секунд)
- Нет выдуманных фактов/дат
- Ключевые слова из вакансии покрыты без спама
- Пункты показывают эффект (цифры, если есть)
- Объём соответствует уровню

## Типовые ошибки
- ИИ “дорисовал” опыт/инструменты
- Слишком общий summary
- Переспам ключевиками
- Нет подтверждаемого эффекта
