# Один текст → 10 постов

Цель: получить **пакет из 10 постов** по одному источнику (те же идеи, разные углы подачи).

## Входные данные (скопируй и заполни)
- Исходный текст (вставь):
- Площадка (X/LinkedIn/Instagram/Telegram):
- Аудитория (кто именно):
- Тон (спокойный/жёсткий/дружелюбный/деловой):
- CTA (что должен сделать читатель):
- Ограничения (без обещаний, без “лечит”, без юр.советов и т.п.):

## Промпт (вставь в ИИ)
**Роль:** ты — редактор контента для соцсетей.  
**Задача:** сделай 10 постов строго по исходному тексту. Не добавляй новые факты. Если тезис не подтверждён исходником — убери или пометь “нужно проверить”.

**Формат результата (10 раз):**
- Пост №:
- Хук (1 строка)
- Тело (3–7 строк)
- CTA (1 строка)
- Хэштеги (0–5, по желанию)
- “Почему работает” (1 строка)

**Правила вариативности:**
- Посты должны различаться: совет, история, чек-лист, миф/факт, ошибка, мини-фреймворк, пример, Q&A, резюме, контрарный тезис.
- Пиши просто. Без хайпа.

**Входные данные:**  
[ВСТАВЬ СЮДА]

## Быстрая проверка качества
- Нет новых фактов сверх источника.
- Каждый пост читается <20 секунд.
- CTA соответствует цели (подписка/сохранить/коммент/скачать).
- Тон единый по всем 10.
