# Заметки → план

**Цель:** Сделать исполнимый план из сырых заметок.

## Что нужно на вход (inputs)
- Сырые заметки
- Дедлайн (если есть)
- Ограничения (бюджет/инструменты/люди)

## Готовые промпты (копипаст)
**Промпт 1 — План из заметок**

```text
Преобразуй заметки в план.
Выход:
- Цель (1 фраза)
- Допущения
- Шаги (нумеровано)
- Ответственные (роли) и сроки (если неизвестно — предложи)
- Риски и меры
Заметки:
<<<NOTES>>>
```

**Промпт 2 — RACI‑лайт**

```text
Сделай таблицу RACI‑лайт: Задача | Responsible | Consulted | Informed.
План:
<<<PLAN>>>
```

**Промпт 3 — Еженедельный статус**

```text
Преврати план в шаблон еженедельного статуса: KPI, блокеры, приоритеты на следующую неделю.
План:
<<<PLAN>>>
```


## Проверка качества (30 секунд)
- Шаги конкретные
- У каждого шага есть ответственный
- Риски реалистичные
- Допущения видны

## Типовые ошибки
- Много шагов без владельцев
- Вообще нет сроков
- Нет раздела рисков
