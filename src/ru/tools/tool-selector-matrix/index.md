# Матрица выбора инструмента

Это **без привязки к брендам**. Помогает выбрать категорию, а не “какой сервис лучше”.

## Быстрая матрица (простая)

| Задача | Лучший тип инструмента | Почему |
|---|---|---|
| Длинные тексты (письма, доки) | Универсальный LLM | Хорошо делает структурный черновик |
| Таблицы и анализ | LLM + таблицы / spreadsheet‑AI | Удобно работать со строками |
| Структура презентации | LLM | Структура + рассказ |
| Генерация картинок | Image‑модель | Визуальный результат |
| Автоматизация | Автоматизация + LLM | Повторяемые конвейеры |

## Промпт (копипаст)
```text
Помоги выбрать инструменты под задачу.
Задача: <задача>.
Ограничения: <бюджет, приватность, скорость>.
Выход:
1) Рекомендуемые категории инструментов (топ‑2)
2) Что проверить за 15 минут (тест‑план)
3) Риски и меры
```
