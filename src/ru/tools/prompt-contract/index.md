# Контракт запроса

“Контракт” — это стабильная форма запроса. Она уменьшает дрейф и повышает повторяемость.

## Контракт (копипаст)
```text
Задача: <что нужно получить>.
Контекст: <зачем / фон>.
Входные данные: <факты, которые ты даёшь>.
Ограничения: <приватность, тон, объём, что обязательно>.
Формат ответа: <разделы/таблица/список>.
Правила качества:
- факты не выдумывать
- неизвестное помечать [CHECK]
- термины не менять: <термины>
```

## Минимальный vs расширенный

### Минимальный (быстро)
```text
Задача: <одно предложение>.
Ограничения: <тон/объём>.
Формат: <буллеты или таблица>.
```

### Расширенный (контроль)
```text
Задача:
Контекст:
Входные данные:
Ограничения:
Выход:
1) Черновик
2) Допущения (если есть)
3) QA-прогон (проблемы + фиксы)
```

## Частые сбои (и как чинить)
- Размытая задача → добавь “Формат ответа”.
- Выдуманные факты → добавь “факты не выдумывать”.
- Слишком длинно → добавь лимит по объёму.
