---
title: Чек-лист качества
---

Лёгкий, повторяемый чек-лист для проверки результата AI-workflow (текст, план, документ, скрипт, датасет) перед выпуском/публикацией.

## Когда применять
Используй перед публикацией, отправкой в команду/клиенту или сохранением как “шаблон / переиспользуемый актив”.

## Гейты качества
### A) Ясность входа
- [ ] Цель задана явно (что считается “хорошо”).
- [ ] Понятны аудитория и контекст.
- [ ] Ограничения указаны (формат, объём, тон, границы).
- [ ] Данные/источники предоставлены (или явно отмечено “нет”).

### B) Контракт выхода
- [ ] Результат соответствует требуемому формату (заголовки, типы списков, нейминг).
- [ ] Все запрошенные части присутствуют (нет пропусков).
- [ ] Соблюдён scope (нет лишнего нерелевантного).
- [ ] Допущения помечены явно (если есть).

### C) Структура и согласованность
- [ ] Одна идея на секцию; нет дублей.
- [ ] Термины используются стабильно (один термин = один смысл).
- [ ] Нумерация стабильная (без “саморасползания”).
- [ ] Таблицы/списки читабельны и соответствуют задаче.

### D) Факты и источники (если есть факты)
- [ ] Факты — либо общеизвестные, либо подкреплены ссылками/источниками.
- [ ] Даты, числа, имена сверены с источниками.
- [ ] Нет уверенных утверждений при нехватке данных.

### E) Безопасность и комплаенс
- [ ] Нет персональных данных (PII).
- [ ] Нет раскрытия конфиденциальной клиентской/внутренней информации.
- [ ] Нет инструкций, ведущих к вреду или нарушению правил.

### F) Готовность к релизу (для переиспользуемых активов)
- [ ] Есть заголовок и короткое описание.
- [ ] Указана версия/статус (например, MVP / v0.1).
- [ ] Работают перекрёстные ссылки (Tools ↔ Pillars ↔ Use cases).
- [ ] Есть паритет EN/RU (для MVP-набора страниц).

## Типовые провалы
- Пропуск части требований или незаметная смена формата.
- Смешивание “допущений” с фактами.
- Поломка ссылок из-за роутинга/относительных путей.
- Дрейф паритета: EN обновили, RU забыли.
- Перегруз “инструкциями” вместо ясного контракта.

## Definition of Done
- Все проверки **A–C** ✅.
- Если есть факты — **D** ✅.
- Если публичная страница — **E** ✅.
- Если это переиспользуемая страница/шаблон — **F** ✅.
