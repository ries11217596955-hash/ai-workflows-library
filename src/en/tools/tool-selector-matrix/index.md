# Tool selector matrix

This is **tool‑agnostic**. It helps choose a category, not a brand.

## Quick matrix (start simple)

| Task | Best tool type | Why |
|---|---|---|
| Long writing (emails, docs) | General LLM | Good at structured drafts |
| Tables and analysis | Spreadsheet AI / LLM + spreadsheets | Handles rows/criteria |
| Slides outline | LLM | Structure + narrative |
| Image generation | Image model | Visual output |
| Automation | Workflow automation + LLM | Repeatable pipelines |

## Copy‑paste prompt
```text
Help me choose tools for this task.
Task: <task>.
Constraints: <budget, privacy, speed>.
Output:
1) Recommended tool categories (top 2)
2) What to test in 15 minutes
3) Failure risks and mitigations
```
