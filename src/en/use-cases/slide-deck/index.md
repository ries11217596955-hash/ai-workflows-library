# Create a slide deck outline

**Goal:** Create a clear presentation outline (10–12 slides).

## What you need (inputs)
- Audience
- Goal
- Key message (1 sentence)
- 3–5 supporting points
- Any data you must include

## Copy‑paste prompt templates
**Prompt 1 — Deck outline**

```text
Create a 10–12 slide outline.
Audience: <audience>
Goal: <goal>
Key message: <one sentence>
Include: slide title + 3 bullets + speaker note.
Constraints: concise, no invented data.
Inputs:
<<<INPUTS>>>
```

**Prompt 2 — One-slide executive summary**

```text
Create a single-slide executive summary: problem, solution, impact, next step.
Context:
<<<CONTEXT>>>
```

**Prompt 3 — Storyboard**

```text
Turn this outline into a storyboard: slide-by-slide narrative and transitions.
Outline:
<<<OUTLINE>>>
```


## Quality check (30 seconds)
- Each slide has one purpose
- No data is invented
- Flow is logical
- Ends with next step

## Common mistakes
- Too many bullets
- No narrative
- Random slide order
